This is a paragraph tag

It puts extra space around the element

It is considered a block level element

This is a span tag. Text in here is put on the same line, unless otherwise indicated. It is considered an inline element.

Void elements such as br and hr don't need closing tags because they don't contain content

img, input, and meta tags are also considered void

Attributes are extra code used to give a tag additional properties or identification information

id="", class="", etc. are a couple of different attributes

They are placed within the opening tag
