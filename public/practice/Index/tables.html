A table tag defines the table

A tr tag defines a row

A th tag defines the heading for the rows and columns

A td tag defines the cells and their contents

The colspan attribute allows you to set how many columns a cell will occupy

The rowspan attrtbute allows you to set how many rows a cell will occupy

The thead, tbody, and tfoot tags seperate the the table and can be individually styled by calling on their respective tags in CSS
